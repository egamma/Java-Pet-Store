Petstore 2.0 Architecture


This document will outline som eof the key choices made for the petstore application, including exporation of various subsystems.

Web application

Chose to make it a pure web application

Patterns to structure application

Used MVC to structure the application

Ajax UI

Usd an Ajax UI so used Dojo and custom JS. Plus decided to create a set of JSF components.

Page is the Application Architecture

How many pages, really just one?

Security

Mashups

Three styles of mashup were used in PS. Proxy, service delegation, and client library

Model Tier
