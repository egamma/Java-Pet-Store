The Java Pet Store Demo is a sample application from the Java 2 Platform, Enterprise Edition BluePrints Program at Java Software, Sun Microsystems. It demonstrates how to use the capabilities of the J2EE 1.4 platform to develop robust and portable enterprise applications.

The Java Pet Store Demo comes with full source code and documentation, which illustrate the typical design decisions and tradeoffs a developer makes when building an enterprise application. The demo shows how to use JavaServer Pages™ ("JSP"), Java™ Servlet, Enterprise JavaBeans ("EJB™"), and Java Message Service ("JMS") technologies.

Note that this version of the Java Pet Store Demo does not include the web service capablity. If you are interested in the webservice functionality on J2EE, refer to the Java Adventure Builder Demo.
