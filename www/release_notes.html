Java Blueprints Petstore 2.0 Release Notes

Changes that are in Early Access Version 0.3

  • Incorporated updated Java Blueprints Solutions Catalog version Early Access 0.7
  • Added Security Filter to make sure only out-facing pages and resources can be accessed without a session
  • Centralized Servlets into a single point of entry which is the ControllerServlet (this still needs to be refactored)
  • Added in Tagging, which includes
    • A tagging display page for top 20 tags to be show in alphabetically order.  Clicking on the tag will show the items associated to the tag.
    • Updated the search page to include a mechanism to add tags to an existing item
    • Updated the seller page to enable tagging of a new item.
  • Updated the Captchas security mechanism on the seller page to use the fileupload component's return status
  • Updated the RSS bar that resides in the Blueprints Solutions Catalog to use Roam


Changes that are in Early Access Version 0.2

  • Updated the Dojo JavaScript library to version 0.3.1
  • Updated to Apache Commons FileUpload library to version 1.1.1
  • Updated to Apache Commons IO library to version 1.2
  • Updated to Apache Commons Logging library to version 1.1
  • Enabled all the example applications to have working Netbeans 5.5 projects as well as command line functionality.
  • Incorporated updated Java Blueprints Solutions Catalog version Early Access 0.6
